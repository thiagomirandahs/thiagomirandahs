### 🚀 Projetos em destaque

#### 🛰️ [**AssetIA**](https://github.com/thiagomirandahs/assetia) — Inventário inteligente de TI com agente LLM
Plataforma full-stack que descobre dispositivos na rede automaticamente (ICMP/ARP/SNMP/OUI) e oferece um **agente Claude com tool use** para responder perguntas em linguagem natural sobre o inventário. Arquitetura **multi-tenant** com JWT + RBAC, scanner async em Python, dashboard React em tempo real e Docker Compose para deploy.
`FastAPI` · `React 19 + TypeScript` · `Claude API` · `Tailwind` · `Docker`

#### 🧑‍💼 [**NexaRH**](https://github.com/thiagomirandahs/nexarh) — Sistema de RH multi-tenant
Plataforma full-stack para gestão de pessoas com **integração de relógio biométrico (Control iD)**, **folha de pagamento conforme CLT**, banco de horas (50%/100%), gestão de férias e benefícios. Arquitetura **SaaS multi-tenant** com isolamento por empresa, **JWT + RBAC**, deploy em VPS Linux com nginx e SSL automático.
`FastAPI` · `React 19 + TypeScript` · `MySQL` · `SQLAlchemy 2.0` · `Tailwind` · `Vite`

#### 🕐 [**Ponto Biométrico Control iD**](https://github.com/thiagomirandahs/ponto-biometrico-controlid) — Ponte LAN ↔ Cloud
Bridge seguro que sincroniza relógios biométricos Control iD na rede local com sistema de RH na nuvem. Lê registros **AFD/Portaria 671**, aplica regras CLT (horas trabalhadas, extras, tolerância) e envia via HTTPS — arquitetura **unidirecional LAN→cloud** para zerar exposição do hardware. **Zero dependências externas** (só stdlib Python).
`Python` · `Control iD REP API` · `Windows Task Scheduler` · `HTTPS/TLS`

#### 🌐 [**Infra & Redes Toolkit**](https://github.com/thiagomirandahs/infra-redes-toolkit) — Automação e endurecimento
Coleção de referência de scripts e configurações para administração e **hardening** de redes corporativas: automação MikroTik (backup, firewall stateful, VLANs), **OpenVPN endurecido** (PKI, TLS 1.2+, AES-256-GCM) e arquitetura segmentada com políticas zone-based **deny-by-default**.
`RouterOS/MikroTik` · `OpenVPN` · `Linux shell` · referência: FortiGate / pfSense / M365 / Entra ID
